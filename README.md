# iNNO FUSION - Premiere Pro Plugin

Motion Graphics Template (MOGRT) Library Manager for Adobe Premiere Pro.

## Features

- Browse and preview MOGRT templates
- One-click import to timeline
- License management system
- Favorites and categories

## Installation

1. Download `iNNO-FUSION.zxp`
2. Install via ZXPInstaller or Anastasiy's Extension Manager
3. Restart Premiere Pro
4. Open from Window &gt; Extensions &gt; iNNO FUSION
